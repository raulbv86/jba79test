# Sistema de Actas — Resp.·. Log.·. "Juan Bautista Arismendi" Nº 79

Aplicación web para la gestión de Trazados de Tenida de la Logia.

## Uso

Abre el archivo `index.html` en cualquier navegador, o accede a la URL de GitHub Pages.

No requiere instalación, servidor, ni conexión a internet para funcionar (solo para descargar el Word).

## Módulos

- ✅ **Módulo 1 — Actas / Trazados** (activo)
- 🔜 Módulo 2 — Convocatorias y Asistencia
- 🔜 Módulo 3 — Reportería y Dashboard
